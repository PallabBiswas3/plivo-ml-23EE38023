This model maximizes the 2,000-step constraint by exploiting the difference between backward passes and optimizer steps, effectively simulating a larger compute budget. We implemented a OneCycleLR scheduler for "Super-Convergence," which optimizes the learning rate trajectory for a fixed-step limit. The architecture is a modernized LLaMA-style GPT, replacing absolute positional embeddings with Rotary Positional Embeddings (RoPE) to support an expanded context window of 256 tokens. Standard layers were replaced with RMSNorm and SwiGLU to improve gradient flow and representation without exceeding the 2,000,000 parameter budget. Finally, we utilized a Char-Byte fallback tokenizer that identifies the 900 most frequent characters (including Devanagari) to compress the sequence efficiently while guaranteeing lossless UTF-8 decoding. Regularization via 5% dropout and label smoothing ensures the model generalizes well to hidden evaluation data, yielding a robust 1.801 BPB.
