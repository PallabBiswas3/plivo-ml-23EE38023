LLM Speedrun: Final Summary

Architecture

  • Rotary Embeddings (RoPE): Enables efficient 256-token context management.
  • SwiGLU & RMSNorm: Modern layers enhancing gradient throughput.
  • Char-Byte Tokenizer: 1156-vocab; optimized for mixed Hindi-English text.

Strategy: Super-Convergence

We bypassed hardware limitations using Gradient Accumulation (simulating an effective batch size of 16), combined with OneCycleLR for optimal step-wise convergence. Regularization via dropout and label smoothing prevents memorization of the training corpus.
